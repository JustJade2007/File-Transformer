# File-Transformer

A locally run application designed for transforming files between various formats without the need for internet connectivity.

## Overview
File-Transformer allows you to convert files, such as Word documents to PDFs and MP4 videos to MP3 audio files, entirely on your local machine. This ensures that your private documents do not need to be uploaded to external servers, providing enhanced security and privacy. It also eliminates dependencies on third-party services, avoiding ads, subscriptions, and paywalls.

## Features
- Fully local file transformations
- Privacy-focused: no data leaves your machine
- Support for common document and media format conversions (e.g., DOCX to PDF, MP4 to MP3)
- No ads or paywalls
